## What is a Runbook?
>A runbook is a step-by-step operational guide used by on-call engineers to **detect, diagnose, mitigate, and recover** from production incidents in a predictable and safe way. 

### Runbooks

This directory contains operational runbooks used during production incidents.

Runbooks help on-call engineers:  
- Respond quickly under pressure  
- Avoid human errors  
- Restore service safely  
- Follow a consistent decision-making process  

### Available Runbooks
- Production Deployment Failure  
- Production Rollback  
- High Error Rate / User Impact
